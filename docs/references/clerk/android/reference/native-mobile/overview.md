# Welcome

Clerk offers native auth, session management, and prebuilt views for your native app.

Use these guides to get started quickly or dive into the SDK reference when you need details.

## Get started

- [Android Quickstart](https://clerk.com/docs/android/getting-started/quickstart.md): Follow the end-to-end setup guide for a Clerk-powered Android app.
- [SDK Reference](https://clerk-android.clerkstage.dev/source/api/index.html): Browse the full Android API reference.

## Explore by feature

- [Authentication flows](https://clerk.com/docs/reference/native-mobile/auth.md): Sign-in, sign-up, and session management.
- [User management](https://clerk.com/docs/reference/native-mobile/user.md): Manage the signed-in user.
- [Organization management](https://clerk.com/docs/reference/native-mobile/organizations.md): Manage members, invitations, and settings.

## Prebuilt views

- [Clerk views](https://clerk.com/docs/reference/views/overview.md): Use Clerk's prebuilt Compose views for fast integration.
- [Customization](https://clerk.com/docs/guides/customizing-clerk/clerk-theme.md): Customize the look and feel of Clerk views.
