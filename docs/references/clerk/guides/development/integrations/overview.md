# Integrations

## Connect with databases and more

- [Vercel Marketplace](https://clerk.com/docs/guides/development/integrations/platforms/vercel-marketplace.md): Configure and integrate Clerk into your Vercel projects through the Vercel Marketplace.
- [Convex](https://clerk.com/docs/guides/development/integrations/databases/convex.md): Clerk offers a first-class integration with Convex, the serverless state management platform.
- [Firebase](https://clerk.com/docs/guides/development/integrations/databases/firebase.md): Clerk integrates directly with Firebase, a popular Platform-as-a-Service (PaaS) provider, so developers can easily add our beautiful Sign Up, Sign In, and User Profile UIs to their Firebase application.
- [Google Analytics](https://clerk.com/docs/guides/development/integrations/analytics/google-analytics.md): Send user authentication events to your Google Analytics property to enable metrics tracking on user sign up and sign in events.
- [Grafbase](https://clerk.com/docs/guides/development/integrations/databases/grafbase.md): Build applications with serverless GraphQL backends provided by Grafbase and user authentication provided by Clerk.
- [Hasura](https://clerk.com/docs/guides/development/integrations/databases/hasura.md): Easily connect to your Hasura GraphQL endpoint using Clerk as your authentication provider.
- [Inngest](https://clerk.com/docs/guides/development/webhooks/inngest.md): Use events from Clerk to trigger functions defined in your codebase with Inngest. Easily handle data synchronization, onboarding campaigns, or billing workflows in code.
- [Instant](https://clerk.com/docs/guides/development/integrations/databases/instantdb.md): Clerk has a default integration with Instant. You get all the auth features of Clerk, and a relational database you can use directly from the browser.
- [Neon](https://clerk.com/docs/guides/development/integrations/databases/neon.md): Build applications using Neon's serverless Postgres database with Clerk as your authentication provider.
- [Nhost](https://clerk.com/docs/guides/development/integrations/databases/nhost.md): The open source alternative to Firebase with GraphQL seamlessly integrates with Clerk to provide authenticated queries.
- [Prisma Postgres](https://clerk.com/docs/guides/development/integrations/databases/prisma-postgres.md): Build applications using Prisma Postgres with Clerk as your authentication provider.
- [Shopify](https://clerk.com/docs/guides/development/integrations/platforms/shopify.md): Use Clerk as your preferred Auth solution for your Shopify store.
- [Supabase](https://clerk.com/docs/guides/development/integrations/databases/supabase.md): Clerk embraces the open source alternative to Firebase and offers a first-class integration with Supabase.
