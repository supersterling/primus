- [Astro](https://clerk.com/docs/reference/astro/overview.md): Easily add secure and SSR-friendly authentication to your Astro application with Clerk.
- [Chrome Extension](https://clerk.com/docs/reference/chrome-extension/overview.md): Use the Chrome Extension SDK to authenticate users in your Chrome extension.

- [iOS](https://clerk.com/docs/ios/reference/native-mobile/overview.md): Use the Clerk iOS SDK to authenticate users in your native Apple applications.
- [Android](https://clerk.com/docs/android/reference/native-mobile/overview.md): Use the Clerk Android SDK to authenticate users in your native Android applications.
- [JavaScript](https://clerk.com/docs/reference/javascript/overview.md): The Clerk JavaScript SDK gives you access to prebuilt components and helpers to make user authentication easier.
- [Nuxt](https://clerk.com/docs/reference/nuxt/overview.md): Easily add secure, beautiful, and fast authentication to Nuxt with Clerk.
- [React Router](https://clerk.com/docs/reference/react-router/overview.md): Easily add secure, edge- and SSR-friendly authentication to React Router with Clerk.

## Backend SDKs

- [JS Backend SDK](https://clerk.com/docs/reference/backend/overview.md): The Javascript Backend SDK exposes our Backend API resources and low-level authentication utilities for JavaScript environments.
- [C#](https://github.com/clerk/clerk-sdk-csharp/blob/main/README.md): The Clerk C# SDK is a wrapper around our Backend API to make it easier to integrate Clerk into your backend.
- [Express](https://clerk.com/docs/reference/express/overview.md): Quickly add authentication and user management to your Express application.
- [Go](https://clerk.com/docs/reference/go/overview.md): The Clerk Go SDK is a wrapper around the Backend API written in Golang to make it easier to integrate Clerk into your backend.
- [Fastify](https://clerk.com/docs/reference/fastify/overview.md): Build secure authentication and user management flows for your Fastify server.
- [Python](https://github.com/clerk/clerk-sdk-python/blob/main/README.md): The Clerk Python SDK is a wrapper around the Backend API written in Python to make it easier to integrate Clerk into your backend.
- [Ruby on Rails](https://clerk.com/docs/reference/ruby/overview.md): Integrate authentication and user management into your Ruby application.
