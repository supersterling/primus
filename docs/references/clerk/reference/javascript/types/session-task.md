# SessionTask

An interface that represents the current [pending task](https://clerk.com/docs/guides/configure/session-tasks.md) of a session.

## Properties

| Name | Type                                      | Description                                                                     |
| ---- | ----------------------------------------- | ------------------------------------------------------------------------------- |
| key  | 'choose-organization' | 'reset-password' | A unique identifier for the task. See the dedicated guide for more information. |
